About Us

The FACE META 2.0 project is a web 3.0 project. Built on Binance Smart Chain. It is a project that includes many different decentralized applications that we call Fapps (Face Meta Dapps). Works with a community focus and acts together with the community, and at the same time aims to distribute its gains back to the community together with Fapps.
